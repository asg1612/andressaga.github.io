Instalación rápida de Glusterfs en Centos 7

Published: jue 11 septiembre 2014
By Andrés Sánchez García

In Sysadmin.

tags: glusterfs centos apunte

En los repositorios por defecto de Centos 7 no viene el servidor glusterfs, con lo que tendremos que instalar manualmente el repositorio.

wget -P /etc/yum.repos.d/ http://download.gluster.org/pub/gluster/glusterfs/3.4/LATEST/CentOS/glusterfs-epel.repo

Una vez añadido el repositorio instalamos el servidor:

yum -y install glusterfs glusterfs-fuse glusterfs-server

Dado que Centos 7 utiliza Systemd tendremos que usar la herramienta systemctl para arrancar el servicio.

systemctl start glusterd

Lo habilitamos para que el servidor gluster se inicie en el arranque.

systemctl enable glusterd

Comprobamos que el servidor gluster ha arrancado correctamente.

systemctl status glusterd

Si todo ha ido bien nos dara la siguiente salida.

glusterd.service - GlusterFS an clustered file-system server
   Loaded: loaded (/usr/lib/systemd/system/glusterd.service; enabled)
   Active: active (running) since jue 2014-09-11 03:37:23 EDT; 8s ago
 Main PID: 17760 (glusterd)
   CGroup: /system.slice/glusterd.service
           └─17760 /usr/sbin/glusterd -p /run/glusterd.pid
